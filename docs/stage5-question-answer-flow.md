# 第5段階 質問・回答待ち・結果確認フロー

## 実装内容

- 手番プレイヤーが「質問する」または「解答する」を選択できる。
- 質問した場合、他プレイヤーが以下から回答する。
  - はい
  - いいえ
  - どちらともいえない
- 回答待ちは最大3分。
- 全員回答または3分経過で回答結果を表示する。
- 回答結果は1分表示する。
- 1分経過後、次プレイヤーへ進む。
- 解答の場合は完全一致で正解判定する。

## 画面仕様

- 自分のお題だけ「？？？」で表示する。
- 自分以外のお題は全員分表示する。
- 質問中は回答状況を表示する。
- 回答結果確認中は各プレイヤーの回答を表示する。

## 起動確認

```bash
cd backend
rm -rf node_modules package-lock.json
npm install
npx prisma generate
npx prisma migrate dev --name stage5_question_answer_flow
npm run start:dev
```

別ターミナル:

```bash
cd frontend
npm install
npm run dev
```

## 手動確認

1. 2人以上でルーム作成・参加する。
2. 全員がお題を提出する。
3. ホストがゲーム開始する。
4. 手番プレイヤーが質問する。
5. 他プレイヤーが「はい / いいえ / どちらともいえない」で回答する。
6. 回答結果が表示される。
7. 1分後に次プレイヤーへ進む。
8. 解答が正解と完全一致したらゲーム終了する。
